## 脚本解析

```console
npm install    :  安装项目对应的依赖
npm run start  :  启动项目，查看DEMO
npm run doc    :  启动文档，依赖vuepress
npm run build  :  根据提示对项目进行打包
npm run cyan   :  对样式库cyan进行打包
npm run maple  :  对方法库和组件库进行打包
npm run theme  :  对主题库进行打包
npm run lint   :  查看代码检查
```
