# list
list组件是非常强大的布局组件，你可以用这个组件来创建任意比例的网格布局。包含两个部分cmui-list和cmui-list-item，其中cmui-list表示外部容器，cmui-list-item表示列表的每一项。

### listGroup的使用
cmui-list组件支持分组操作，并提供索引功能，借助这一特性，你可以很方便的创建诸如音乐播放列表，手机通讯录，城市选择等形式的列表，并在屏幕右侧提供索引
